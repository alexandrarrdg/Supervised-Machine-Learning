# Supervised Machine Learning

Notebooks prácticos sobre fundamentos de Machine Learning supervisado, con un enfoque en **adopción responsable de IA en organizaciones**.

## Objetivo

Esta serie explica conceptos clave de ML supervisado de forma progresiva: desde los modelos más simples hasta técnicas más avanzadas. Cada notebook combina **teoría, código desde cero, implementación con scikit-learn, y reflexiones sobre el impacto real** de estos modelos cuando salen del notebook y entran en el mundo real.

No se trata solo de aprender a usar herramientas, sino de entender qué estamos construyendo, qué patrones capturan los modelos, y qué implicaciones tienen las decisiones técnicas que tomamos.
